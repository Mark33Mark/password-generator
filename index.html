Password generator - different scripting methods

Array or String?

I've quickly put this site together so that you have one website to open either process I've explored with coding the password generator. Please note that either option you select presents and operates identically.

Providing you with 2 options is to demonstrate that I've explored 2 coding possibilities and then looked at if there was any processor performance benefits of one over the other.

I found that on average, the string method is approximately 8% faster. I wasn't expecting this as you should expect array process to be more efficient.

Please refer to the README file in my GitHub repository that has a table summarising the processing speed results.
